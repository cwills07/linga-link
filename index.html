Project Overview

Traditional language learning applications, while providing structure and organization, often fall short in replicating the engagement and spontaneity of real-world conversation. This lack of practical application hinders effective language acquisition and fluency development. Learners struggle to transition from memorized phrases to natural conversation, leading to frustration and a plateau in their progress.

Lingua-Link addresses this challenge by creating a secure peer-to-peer language exchange community through a mobile application. This innovative app fosters meaningful conversation and cultural exchange between learners of different languages.

Research & Design Process

1. Discovery

Goal: Understand user perceptions and attitudes toward different language learning platforms

    Methodology:
  • Survey
  • Competitive Analysis
  • User Interviews + Usability Testing

2. Exploratory

Goal: Find opportunities for improvement on lingua-link’s features and functionalities

    Methodology:
  • Prototype feedback and testing on lingua-link prototype
  • User journey mapping

3. Testing/Iteration

Goal: Find any user pain points and get feedback on lingua-link’s privacy and security features

    Methodology:
  • Usability testing on lingua-link's MVP

4. Handoff

Goal: Further develop and implement lingua-link as a safe and secure peer-to-peer language learning application

    Methodology:
  • Beta testing
  • Incorporating designer and developer feedback

Key Features

Secure Onboarding

  • User verification through 2FA and ID upload
  • Customizable avatars for users
  • Community guidelines

Verified User Matching

  • Users must approve lingua-link matches
  • AI content and chat moderation
  • Flagging and blocking users for inappropriate behaviors

Personalized Language Learning

  • Users can practice lessons together
  • Customizable games and interactive lessons
  • Interactive progress tracking
